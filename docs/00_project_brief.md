¿Qué es EdificiOS?

EdificiOS no es un gestor de tickets.

Es un Sistema Operativo para la operación diaria de edificios.

El dominio modela trabajo operativo.

No modela conversaciones.

No modela organigramas.

La unidad principal de trabajo es el Turno.

El dominio está compuesto por agregados independientes (Incident, Custody, Inspection, Maintenance).

El conocimiento pertenece a la organización.

La fuente de verdad son los eventos.

La interfaz está diseñada para registros inferiores a 15 segundos.
